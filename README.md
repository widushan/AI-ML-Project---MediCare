# MediCare

A Machine Learning powered healthcare assistant that provides medicine recommendations, symptom analysis, and health resources.

## Features
- Medicine recommendation system
- Symptom severity analysis
- Health articles and resources
- Diet and workout suggestions

## Project Structure
- `main.py`: Main application entry point
- `datasets/`: Contains CSV files for symptoms, medications, diets, etc.
- `models/`: Pre-trained ML models and notebooks
- `templates/`: HTML templates for web interface
- `static/`: Images and static assets

## Installation
1. Clone the repository:
   ```powershell
   git clone https://github.com/widushan/AI-ML-Project---MediCare.git
   ```
2. Set up a Python virtual environment:
   ```powershell
   python -m venv venv
   .\venv\Scripts\activate
   ```
3. Install required packages:
   ```powershell
   pip install -r requirements.txt
   ```

## Usage
Run the main application:
```powershell
python main.py
```
